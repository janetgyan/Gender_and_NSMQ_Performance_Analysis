# Gender_and_NSMQ_Performance_Analysis

## Overview

This project analyzes NSMQ (National Science & Maths Quiz) exam performance across different gender groups to identify performance patterns, strengths, and areas for improvement. The insights support data-driven strategies for student preparation and academic performance analysis.

## Objectives

* Analyze exam performance trends across male and female participants
* Identify differences in performance based on gender
* Understand factors contributing to performance outcomes
* Provide actionable insights through interactive dashboards

## Dataset

*NSMQ exam scores for multiple years
* Includes gender, school, region, and total scores
* Data was cleaned and prepared for analysis

## Tools Used

* Power BI (data modeling & dashboard creation)
* Excel (data cleaning and preparation)

## Key Insights

* Male and female participants show slight variations in average scores across years
* Certain regions consistently produce higher-performing teams
* Performance distribution highlights areas where targeted support could improve results

## Visualizations

The Power BI dashboard includes:

* Gender-based score comparison
* Performance distribution by school and region
* Year-on-year performance trends
* High and low scoring areas by gender

## Conclusion

The analysis reveals performance trends across gender and highlights strengths and weaknesses among NSMQ participants, providing actionable insights for educators, coaches, and stakeholders.

## Files in This Repository

* `powerbi/` – Power BI dashboard file (.pbix)
* `data/` – Dataset used for the analysis
* `visuals/` – Dashboard screenshot
* `reports/` – Summary report
